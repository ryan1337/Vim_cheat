Vim cheat sheet en français

  • La base:

  1. Le curseur ce déplace avec les touche hjkl

    2. gauche ==> touche h

    bas ==> touche j

    haut ==> k

    droite ==> l

  2. (Echap :q! Entrée): Quitte Vim sans enregistrer.
  3. (Echap :wq Entrée), (Echap :x Entré), (ZZ): Quitte Vim en enregistrant.
  4. x: Efface un caractère en mode normal.
  5. i: Active le mode insertion
  6. Echap: Quitte le mode insertion
