Vim cheat sheet en français

  • La base.
  • Supressions, Annulations.
  • Insertions.
  • Couper, Copier, Coller du texte.
  • Recherche, Positions.
  • Commande externe.

  1. La base.

    • Le curseur ce déplace avec les touche hjkl

      • gauche ==> touche h

      bas ==> touche j

      haut ==> k

      droite ==> l

    • (Echap :q! Entrée): Quitte Vim sans enregistrer.
    • (Echap :wq Entrée), (Echap :x Entré), (ZZ): Quitte Vim en enregistrant.
    • x: Efface un caractère en mode normal.
    • i: Active le mode insertion.
    • Echap: Quitte le mode insertion.

  2. Supressions, Annulations.

    • dw: Efface du curseur jusqu'à la fin du mot.
    • d$: Efface du curseur jusqua'à la fin de la ligne.
    • dd: Efface toute une ligne.
    • u: Efface les actions precedente.
    • U: Annule tout les changement sur une ligne.

  3. Insertions.

    • o: Ouvre une ligne en mode insertion sous le curseur.
    • O: Ouvre une ligne au dessus du curseur.
    • a: Insère du texte après le caractère où ce trouve le curseur.
    • A: Ajoute du texte a la fin de la ligne.
    • r: Remplace le caractère sur le curseur.
    • R: Active le mode remplacement.

  4. Couper, Copier, Coller du texte.

    • dd: Coupe le texte.
    • p: Colle le texte après la position courante.
    • P: Colle le texte avant ou de au dessus de la position courante.

  5. Recherche, Positions.

    • Ctrl-G: Affiche la position et l'etat du fichier.
    • G: Place a la fin du fichier.
    • /: Recherche vers l'avant.
    • ?: Recherche vers l 'arrière.

  6. Commande externe.

    • :!command - Execute une commande externe.
    • :!ls - Affiche le contenu du dossier courant.
    • :!rm "fichier" - Efface "fichier".
    • :w "fichier" - Enregistre le fichier.
